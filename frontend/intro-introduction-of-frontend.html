# 开始之前

在正式学习前端相关知识之前，请确保自己满足以下条件：

  1. 安装好 Google Chrome在新窗口打开 / Microsoft Edge在新窗口打开 / Mozilla Firefox在新窗口打开 浏览器，而不是使用各种国产浏览器（如 360、2345、搜狗）。
  2. 安装任意你喜欢的代码编辑器，如 Visual Studio Code在新窗口打开、HBuilder X在新窗口打开、Sublime Text在新窗口打开（收费）、JetBrains WebStorm在新窗口打开（收费）。
  3. 尽量有够访问国际互联网。
