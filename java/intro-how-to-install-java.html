# 如何安装 Java

在 Java 世界中，有不同厂商提供的 JDK 可供使用。例如OpenJDK在新窗口打开、Oracle JDK在新窗口打开、Azul Zulu JDK在新窗口打开、Temurin（原 AdoptOpenJDK）在新窗口打开等等。他们的使用协议各不相同，有些商业用途需要付费。

作为开发人员，我们使用最多的则为Oracle JDK在新窗口打开，本文也以 Oracle JDK 安装作为演示。

# Windows 平台

# JDK 下载

进入Oracle JDK在新窗口打开官网，切换到 Windows 选项卡，选择合适的版本进行下载。

当然，如果你嫌麻烦的话呢，可以直接点击下方链接进行下载。

  • 压缩包在新窗口打开
  • 安装包在新窗口打开
  • MSI 包在新窗口打开

下载完成后解压/安装到某个目录中，如C:\Develop\jdk-17

# 环境变量配置

  1. 添加JAVA_HOME，指向C:\Develop\jdk-17
  2. 在Path变量中加入%JAVA_HOME%\bin

# macOS 平台

# JDK 下载

  • ARM 版压缩包在新窗口打开
  • ARM 版安装包在新窗口打开（推荐）
  • x64 版压缩包在新窗口打开
  • x64 版安装包在新窗口打开（推荐）

注意：

  1. 使用 ARM 架构芯片的 MacBook（如 M1）请选择 ARM 版 JDK。
  2. 使用压缩版的 JDK 需要将其解压到合适的目录，如~/Develop/jdk-17

# 环境变量配置

用任意文本编辑器打开~/.bash_profile或~/.zprofile，在最后一行加入以下内容：

export JAVA_HOME="/Users/[用户名]/Develop/jdk-17" # 压缩版请使用这一条
export JAVA_HOME=`/usr/libexec/java_home -v 17`  # 安装版请使用这一条
export PATH=$JAVA_HOME/bin:$PATH

# 安装后的检测

打开系统上的命令行（如 Windows 中命令提示符，macOS中 Terminal），输入java -version，如能正确显示 JDK 版本信息，则证明 JDK 安装、环境变量配置成功。

$ java -version
java version "17" 2021-09-14 LTS
Java(TM) SE Runtime Environment (build 17+35-LTS-2724)
Java HotSpot(TM) 64-Bit Server VM (build 17+35-LTS-2724, mixed mode, sharing)
